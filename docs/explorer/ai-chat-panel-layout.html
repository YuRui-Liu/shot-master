④ 更新后，各段 Inspector 的 prompt 实时反映新方向；你逐段试听旧候选或点「生成」出新候选。

数据流：指令 → 文本 LLM（复用 deepseek/refine 配置）→ 配乐方向 JSON → 写入 session 各段 music_prompt → 持久化。不触发 RunningHub。
