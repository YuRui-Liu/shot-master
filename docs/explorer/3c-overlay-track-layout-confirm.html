顺带确认的另外两个决策（任选，可在选布局时一起告诉我）

① 折叠 / 高度策略
  • 可折叠 + 动态高度（推荐）：叠加区带标题行整体折叠，空时隐藏
  • 总是展开：实现最简，lane 多时占满高度
Option A 自带折叠头；Option B 折叠需拆两段。
② 片段交互范围（完整框选生成在 3d）
  • 只读渲染 + 可选中（推荐）：点选→Inspector 看 prompt/音量，本阶段不拖拽/缩放
  • 渲染 + 拖拽缩放：要扩 commands/undo 栈支持 overlay，scope 变大
  • 纯只读：不可点，Inspector 看不到 overlay 细节
③ overlay 子轨头部控件
  • 每 lane 一个 M + 音量（推荐）：与固定轨一致的轻量控件
  • 纯标签：只显示 bgm·0 名字，控件留到 3d
