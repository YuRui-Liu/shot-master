快速开始

五步跑通第一次操作：选目录、设输出、选功能、填参数选图、执行。本页帮你在几分钟内完成从打开软件到产出第一个结果。

系统要求

  • 操作系统：Windows 10 / 11（64 位）。
  • 账号：使用视频生成、配音、配乐功能需要 RunningHub 账号与 API Key。
  • 网络：生成类功能（视频 / 配音 / 配乐）依赖云端，需要联网。
  • 本地功能：图像处理（拆图 / 拼图 / 去白边）在本地完成，无需联网。

安装与启动

  1. 解压下载得到的发行包到任意目录。
  2. 运行目录中的主程序 exe 启动软件。
  3. 首次启动需要激活，按提示完成激活后即可使用（详见 激活与授权）。
若启动时被杀毒软件拦截或误报，请参阅 故障排查 的相关说明。

五步跑通

  1. 在左栏点「打开目录」，选择存放分镜图的文件夹。
  2. 点「设置输出目录」，选择处理结果的导出位置。
  3. 在顶部功能切换条选择要用的功能（例如「拆图」）。
  4. 在右栏填写参数，在中栏缩略图区选择要处理的图片。
  5. 点「执行」开始处理；视频 / 配音 / 配乐功能则在对应的任务窗口点「生成」。

提示

中栏缩略图大小可调，便于浏览大量分镜；选图时支持用 Ctrl 点选多张、用 Shift 选择连续区间。
