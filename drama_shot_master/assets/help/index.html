Drama-Shot-Master

面向短剧分镜的桌面工作台：图像处理（拆图 / 拼图 / 去白边）+ 基于 LTX 2.3 的视频生成与配乐，一站式完成从分镜图到成片的流程。

顶部功能切换条分两组——图像（拆图 / 拼图 / 去白边）与 视频（视频生成 / 配乐）。左栏管理当前目录与输出目录，中栏是缩略图， 右栏是当前功能的参数面板。

本帮助文档随软件持续完善，后续会补充更详细的操作说明与示例。

拆图 图像处理

把一张网格大图（如 2×2、4×4 分镜）自动切成多张子图。基于白带检测定位 每个格子，对不均匀间距/外边距鲁棒。

  • 设置源网格行列数与目标子图行列数。
  • 支持「一键检测」白边/网格，自动填参数。
  • 可选重采样后处理（Lanczos / AI 放大）。

拼图 图像处理

把多张图按 R×C 网格拼接成一张大图。用 order 模式逐个点选确定 排列顺序（按 Shift 可区间选，按行序追加）。

  • 可设目标比例、间距、缩放方式（留边 / 裁切 / 拉伸）。

去白边 图像处理

自动迭代裁掉图片四周残留的白边。

  • 可调阈值与最大迭代次数。
  • 白边去不干净时，可用「额外向内裁剪（上/下/左/右）」手动微调。
  • 选图支持 Ctrl+点击多选、Shift 区间选。

视频生成 视频

基于 LTX 2.3 导演台工作流，从分镜图 + 提示词生成视频。支持多任务并行—— 每个任务独立窗口，不满意可回到对应窗口重生成，无需重拖轨道。

  • 拖拽分镜到时间轴，分段写提示词（运镜 / 画面 / 音效）。
  • 可切换「导演台」与「ALL IN ONE V3」工作流。
  • 提交前需在 RunningHub 配置中填好凭据。

配乐 视频

为成片自动分析卡点并生成/对齐配乐，独立任务窗口管理。

  • 在「设置 → 配乐…」配置相关参数。

配音 视频

基于 RunningHub TTS 工作流为台词配音，独立任务窗口管理。任务窗顶部切换两种方式：

  • 音色设计：填「要合成文本 + 音色描述（自然语言）+ 语言」，由模型按描述生成音色。
  • 声音克隆：填「文本 + 说话人参考音频 + 情感强度」，并按需选 4 种情感模式之一—— 默认（随参考音频）/ 文本情绪（写情感描述）/ 语音情绪模仿（传情感参考音频）/ 情感向量 （8 维 Happy/Angry/Sad/Fear/Hate/Low/Surprise/Neutral）。
  • 生成完点「打开结果」试听 FLAC。
  • 在「设置 → 配音…」配置两个工作流 ID 与输出目录。

自定义工作流 高级

本软件不在本地跑模型，而是把任务提交到 RunningHub 上你已保存的工作流 （按 workflow_id），并用「节点参数覆盖」把界面里的输入写进工作流对应节点。 因此「接入自定义工作流」= 在 RunningHub 部署你的工作流 + 在软件里填它的 ID + 必要时校准节点号。

一、换用你自己的工作流（无需改代码）

  1. 在 RunningHub 部署/保存你的工作流，复制它的 workflow_id。
  2. 视频：「设置 → RunningHub 配置」里，每个工作流（导演台 / ALL IN ONE V3）各有一个 workflow_id 输入框，填入即可；还可设「自定义模板路径」指向你导出的模板 JSON。
  3. 配音：「设置 → 配音…」里填「音色设计 / 声音克隆」两个 workflow_id。
  4. 提交前确保「RunningHub 配置」里 API Key、Base URL 正确。

二、节点号不一致时（覆盖映射）

软件靠固定的「角色→节点号」映射把输入写进工作流。若你的工作流节点号与默认不同， 编辑配置文件 settings.json 校准即可（无需改代码）：

  • 配音：在 dub_node_profiles 下按工作流（voice_clone / voice_design）填「角色:节点号」覆盖。例如声音克隆默认 文本=4、说话人音频=10、情感描述=16、情感参考音频=19、情感向量=21、模式选择器=26、 四个分支=1/14/17/20。
提示：导出 RunningHub 工作流的 API 格式 JSON，对照其中每个节点的 class_type 与输入名，即可确认要覆盖的节点号。

三、新增一种工作流类型（开发者向，需改代码）

要让软件支持一类全新工作流（新的节点结构 / 新的可调项），在代码里登记一个 profile：

  • 视频：在 drama_shot_master/core/workflow_profiles.py 的 PROFILES 增加 WorkflowProfile（含模板文件名 + 各节点号），把模板 JSON 放进 drama_shot_master/templates/。
  • 配音/TTS：在 drama_shot_master/core/tts_profiles.py 增加 TTSProfile （workflow_id + 角色→节点号），并按需扩展提交构造器。
无论哪种方式，软件只覆盖工作流里的 widget / 开关，不修改工作流图本身； 多分支选择（如 rgthree 选组节点）通过覆盖其组开关实现。

激活与授权 其它

软件需激活后使用。首次启动会要求输入激活码。

  1. 在「关于」对话框复制本机机器码，发给作者。
  2. 作者签发与该机器绑定、带有效期的激活码发回。
  3. 把激活码粘进「关于」的输入框点激活即可。到期前会提示续期。
激活码与本机绑定，仅在本机生效；到期后需重新激活。
