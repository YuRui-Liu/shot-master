故障排查

ComfyUI / RunningHub 不可达

  • 检查本机网络是否正常、能否访问外网。
  • 确认「设置 → RunningHub 配置」里的 API Key 与 Base URL 填写正确、无多余空格。
  • 私有部署时确认服务地址与端口可达。

生成失败 / 报错

  • 先看任务窗内的状态与错误提示。
  • 确认对应工作流的 workflow_id 填写正确。
  • 确认所需素材（分镜图 / 参考音频等）已正确选好并上传。

视频「不遵循图 / 分辨率错」

  • 确认选用了正确的工作流类型。
  • 多为节点映射不匹配所致，参考 自定义工作流 校准节点号。
若结果完全偏离分镜图，优先核对工作流选择与节点映射，再调整提示词。

激活失败

  • 提示「激活码非本机」：机器码不匹配，请用本机「关于」对话框里的机器码重新申请。
  • 提示「激活码无效」：多为复制不全或含空格，重新完整复制激活码再试。

启动被杀毒软件拦截（打包版）

  • 原生编译的可执行文件常被误报，属正常现象。
  • 在杀毒软件中将程序加入信任 / 白名单，或使用已签名的发行版本。

找不到输出文件

  • 确认已在「设置」里为对应功能配置了输出目录。
  • 到该输出目录及其子目录下查找视频 / 配音 / 配乐文件。
