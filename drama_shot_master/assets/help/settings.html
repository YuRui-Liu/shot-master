设置说明

概述

菜单栏「设置」下集中了软件各项配置对话框。生成类功能（视频 / 配乐 / 配音）依赖云端 RunningHub，使用前需先在这里填好账号信息与各工作流 ID。本页逐个说明各对话框的字段含义。

RunningHub 配置

视频生成的云端接入配置，也是其它生成功能的账号基础。

字段含义
API KeyRunningHub 账户的访问密钥，用于鉴权与提交任务。
Base URLRunningHub 服务地址，一般保持默认，私有部署时按实际填写。
视频输出目录生成的视频文件保存到的本地文件夹。
各工作流 workflow_id每种视频工作流（导演台 / ALL IN ONE V3 等）在 RunningHub 上对应的已保存工作流 ID。
自定义模板路径指向自己导出的工作流模板 JSON，留空则使用内置模板。

翻译配置

提示词等文本的翻译服务配置。

字段含义
翻译服务所用翻译后端（如 DeepLX 等）。
服务地址翻译服务的接口地址。
密钥 / Token若所选服务需要鉴权，则在此填写。
具体可填字段以软件内对话框为准；不确定的字段保留默认即可。

提示词优化配置

用大模型按 meta-prompt 优化、扩写视频提示词的配置。

字段含义
meta-prompt 路径指向优化所用的元提示词文件。
Provider / base_url大模型服务地址（如 Ollama 本地服务或豆包等在线服务）。
API Key所选服务的访问密钥（本地服务可留空）。
Model使用的模型名称。

配乐

自动配乐相关的默认参数。

字段含义
风格生成或匹配配乐的风格倾向。
随机种子控制结果可复现的随机种子。
交叉淡入淡出片段衔接处的淡入淡出时长。
卡点参数卡点分析的灵敏度、最大数量等。

配音

配音（TTS）工作流的接入配置。

字段含义
音色设计 workflow_id「音色设计」方式在 RunningHub 上对应的工作流 ID。
声音克隆 workflow_id「声音克隆」方式在 RunningHub 上对应的工作流 ID。
输出目录生成的配音音频（FLAC）保存到的本地文件夹。
当线上工作流的节点编号与默认映射不一致时，可在配置文件 settings.json 的 dub_node_profiles 中做高级覆盖，详见 自定义工作流。
