常见问题

账号与配置

如何获取 RunningHub API Key？

登录 RunningHub 账户，在账户 / 控制台页面找到 API Key 并复制，填进「设置 → RunningHub 配置」。视频生成、配乐、配音都依赖它。

图像处理要联网吗？

不需要。拆图 / 拼图 / 去白边都在本地完成，无需联网；视频生成、配乐、配音是云端任务，需要联网与 RunningHub 账号。

生成任务

生成一个视频 / 配音要多久？

取决于云端排队情况与目标时长，没有固定值。提交后可在对应任务窗内查看实时状态。

能同时跑多个任务吗？

能。视频 / 配乐 / 配音都是任务式面板，每个任务占用独立窗口并行运行，互不干扰。

输出结果

生成的文件保存在哪里？

在「设置」里为各功能配置的输出目录下。视频、配音（FLAC）、配乐分别存放在各自的输出目录 / 子目录中。
