视频生成

概述

视频生成基于 LTX 2.3 导演台工作流，从分镜图加提示词生成带运镜的视频片段。每个生成任务是一个独立窗口，可同时开多个并行处理。

适用场景

把已经处理好的分镜图变成带运镜、有节奏的视频片段，用于拼成短剧成片。

操作步骤

  1. 在顶部功能条选择「视频生成」，点「新建」开启一个独立任务窗口。
  2. 把分镜图拖到任务窗的时间轴上，按片段排好顺序。
  3. 为每个分段填写提示词，描述运镜、画面内容与音效。
  4. 选择要使用的工作流（如「导演台」或「ALL IN ONE V3」）。
  5. 点「生成」提交任务，完成后在窗口内预览结果。

多个任务可以并行运行，各自占用独立窗口互不干扰；若某个片段不满意，回到对应窗口调整提示词后重新生成即可。

参数说明

参数含义
全局提示词对整段视频统一生效的画面 / 风格描述。
时长 / 帧数生成视频的总长度与帧数设置。
分段长度每个分镜片段的时长，决定节奏与切换。
引导强度提示词与分镜图对生成结果的约束强度。
是否自定义音频是否使用自带音频；关闭时由工作流生成原生音频。
工作流选择选用的 LTX 2.3 工作流类型（导演台 / ALL IN ONE V3 等）。

提示与技巧

提交生成前，需先在「设置 → RunningHub 配置」中填好 API Key 与各工作流的 workflow_id，否则任务无法提交到云端。

常见问题

生成结果不遵循分镜图 / 分辨率不对怎么办？

多为工作流选择或节点映射问题，请参阅 故障排查。

想换用自己的工作流怎么办？

见 自定义工作流，介绍如何填入自己的 workflow_id 与节点覆盖。
