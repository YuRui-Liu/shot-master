## 为每个镜头提供
- `speaker`：说话人（角色名 / "旁白" / "-"（无台词））
- `dialogue`：台词文本（从剧本提取，若无则"-"）
- `sfx`：音效描述（如"雨声·轻盈"，若无则"-"）
- `bgm_emotion`：从以下选择：悲切·慢板 / 温柔·中板 / 静默·留白 / 紧张·快板 / 喜悦·轻快 / 神秘·空灵

## 输出（严格 JSON）
```json
{{
  "cues": [
    {{
      "shot_id": "S01",
      "speaker": "旁白",
      "dialogue": "世人皆说守株待兔是愚…",
      "sfx": "雨声·轻盈",
      "bgm_emotion": "悲切·慢板"
    }}
  ]
}}
```

规则：
- shot_id 与输入完全一致
- 台词要从剧本提取，不要发明台词
- 不要在 JSON 外输出任何内容
